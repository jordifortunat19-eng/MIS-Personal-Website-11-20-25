## Getting Started

1. **Add Your Content**
   - Replace all placeholder text (marked with `[brackets]`) with your personal information
   - Add your profile picture to `images/profile.jpg`
   - Add hobby images to `images/hobby1.jpg`, `hobby2.jpg`, etc.
   - Add your welcome video to `videos/welcome.mp4`
   - Add the team UMD video to `videos/umd-video.mp4`
   - Add your PDF resume to `files/resume.pdf`

2. **Customize the Design**
   - Edit `css/style.css` to change colors, fonts, or layout
   - Color scheme is defined in CSS variables at the top of `style.css`

3. **Update the Game**
   - Replace the placeholder game in `js/game.js` with your team's actual game
   - The game should be engaging and demonstrate JavaScript skills

4. **Test Your Website**
   - Open `index.html` in a web browser
   - Test all navigation links
   - Verify responsive design on mobile devices
   - Check that all images and videos load correctly

## Customization Guide

### Changing Colors
Edit the CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #2c3e50;
    --secondary-color: #3498db;
    --accent-color: #e74c3c;
    /* ... */
}
```

### Adding New Pages
1. Create new HTML file
2. Copy the header/navigation structure from existing pages
3. Add link to navigation menu in all pages
4. Style the new page in `style.css`
